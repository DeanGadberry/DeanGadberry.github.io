Git Resources

Karson, thank you for reaching out and asking about GitHub. I am excited to share these compiled resources with you and anyone else who is interested in learning about these great tools.

Do this and you'll Git it.

  1. Write down a short-term goal.
    • What do you want to be able to complete now?
  2. Write down a long-term goal.
    • What do you want to be able to do in a month or year?
  3. Watch Git Explained in 100 Seconds.
  4. Create a list of words that you look up in The GitHub Glossary while you read and watch the following.
    • About Git
    • Git Explained: The Basics
    • The essentials of Git explained in five minutes
  5. Try it out with the GitHub Docs "Hello World" Guide
  6. Watch Git Explained in 100 Seconds again.
